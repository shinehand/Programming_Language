# 그래프 (Graph)

## 개념

그래프는 **정점(Vertex)**과 **간선(Edge)**으로 구성된 자료구조입니다.

- **무방향 그래프**: 간선에 방향이 없음 (양방향)
- **방향 그래프**: 간선에 방향이 있음
- **가중치 그래프**: 간선에 가중치(비용)가 있음

## 표현 방법

### 인접 행렬 (Adjacency Matrix)

```
     0  1  2  3
  0 [0, 1, 1, 0]
  1 [1, 0, 1, 1]
  2 [1, 1, 0, 1]
  3 [0, 1, 1, 0]
```

- **장점**: 간선 존재 여부 O(1) 확인
- **단점**: 메모리 O(V²), 희소 그래프에 비효율적

### 인접 리스트 (Adjacency List)

```
0: 1(w=5) → 2(w=3) → NULL
1: 0(w=5) → 2(w=2) → 3(w=4) → NULL
2: 0(w=3) → 1(w=2) → 3(w=1) → NULL
```

- **장점**: 메모리 O(V+E), 이웃 순회 효율적
- **단점**: 간선 존재 여부 O(degree) 확인

## 비교

| 특성 | 인접 행렬 | 인접 리스트 |
|---|---|---|
| 메모리 | O(V²) | O(V+E) |
| 간선 확인 | O(1) | O(degree) |
| 이웃 순회 | O(V) | O(degree) |
| 적합한 경우 | 밀집 그래프 | 희소 그래프 |
