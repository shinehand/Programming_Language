# BFS / DFS 탐색

## BFS (너비 우선 탐색)

시작 정점에서 **가까운(같은 레벨) 정점부터** 차례로 탐색합니다. **큐(Queue)**를 사용합니다.

```
     0
    / \
   1   3
   |   |
   2   4
       |
       5

BFS(0): 0 → 1 → 3 → 2 → 4 → 5
```

### 특징
- 최단 경로 보장 (가중치 없는 그래프)
- 레벨 순서로 탐색

## DFS (깊이 우선 탐색)

한 방향으로 **최대한 깊이** 탐색 후 백트래킹합니다. **스택(Stack)** 또는 **재귀**를 사용합니다.

```
DFS(0): 0 → 1 → 2 → (백트래킹) → 4 → 3 → 5
```

### 특징
- 경로 탐색, 미로 탈출
- 사이클 감지
- 위상 정렬

## 비교

| 특성 | BFS | DFS |
|---|---|---|
| 자료구조 | 큐 (Queue) | 스택 / 재귀 |
| 최단 경로 | ✅ (가중치 없음) | ❌ |
| 메모리 | 더 사용 | 덜 사용 |
| 적합한 경우 | 최단 경로, 레벨 탐색 | 경로 탐색, DFS 순회 |

## 컴파일 및 실행

```bash
g++ -std=c++17 bfs_dfs/bfs.cpp -o bfs && ./bfs
g++ -std=c++17 bfs_dfs/dfs.cpp -o dfs && ./dfs
```
